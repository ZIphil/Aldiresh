ゲームの説明

水色のオブジェクトがプレイヤーで、桃色のオブジェクトが敵です。

敵に弾を 3 回当てることで敵を倒すことができます。

逆に敵から撃たれた弾に当たるとライフが減り、ライフが 0 になるとゲームオーバーになります。

黄色のオブジェクトはアイテムで、触ることでライフを回復することができます。

命中率が高いほどスコアが高くなります。

短時間に連続で的に弾を当てると、コンボボーナスが発生してスコアが高くなります。
